# Topology Modes

This example demonstrates Rabbit Relay topology modes.

It shows:

- app-owned topology with `topologyMode: "assert"`
- infrastructure-owned topology with `topologyMode: "passive"`
- CI / review mode with `topologyMode: "plan-only"`
- how topology modes work with `planTopology()`
- how topology modes relate to delayed retry topology

Full example on GitHub:  
https://github.com/bitspacerlabs/rabbit-relay/tree/main/examples/16-topology-modes
